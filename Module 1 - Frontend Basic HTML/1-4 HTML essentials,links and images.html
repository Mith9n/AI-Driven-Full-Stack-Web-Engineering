HTML Links Links are used to connect one webpage to another webpage, website, file, or location. HTML uses the <a> (anchor) tag. Basic Link Syntax: <a href="URL">Link Text</a> Example: <a href="https://www.google.com"> Visit Google </a> Output: Visit Google (clickable link) Types of Links: 1. External Link Link to another website: <a href="https://www.facebook.com"> Facebook </a> 2. Internal Link Link to another page: <a href="about.html"> About Us </a> 3. Open Link in New Tab Use target="_blank" <a href="https://www.google.com" target="_blank"> Google </a> 4. Email Link <a href="mailto:example@gmail.com"> Send Email </a> 5. Download Link <a href="file.pdf" download> Download File </a> -------------------------------------------------- 3. HTML Images Images are used to display pictures on webpages. HTML uses the <img> tag. Basic Image Syntax: <img src="image.jpg"> Example: <img src="photo.jpg"> Image Attributes: 1. src Attribute Defines image location. Example: <img src="nature.jpg"> 2. alt Attribute Shows alternative text if image cannot load. Example: <img src="photo.jpg" alt="My Photo"> 3. Width and Height Changes image size. Example: <img src="photo.jpg" width="300" height="200"> -------------------------------------------------- Complete HTML Example: <!DOCTYPE html> <html> <head> <title>HTML Links and Images
